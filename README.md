# EcoSpark (Repository Root)

This repository contains the EcoSpark Django application. The main application is in the `EcoSpark/` directory — that folder is the Django project root and contains the app `core/`.

See `EcoSpark/README.md` for full setup and developer instructions. This top-level README provides a short overview.

Short overview

- Project: EcoSpark — E‑Waste Finder
- Tech: Python, Django, Bootstrap, minimal JavaScript
- Purpose: Help users find recycling centers, learn about e‑waste, estimate recoverable value, schedule pickups, and engage with gamified challenges.

Quick start

1. Change into the project folder and follow the detailed README:

```powershell
cd EcoSpark
code .
# then follow EcoSpark/README.md steps
```
